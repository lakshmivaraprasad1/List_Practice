🛢️ What is MySQL?
    MySQL is an open-source relational database management system (RDBMS) used to store, organize, and manage data using SQL (Structured Query Language).

🔧 Key Features: 
  • Supports structured data with tables, rows, and columns
  • Allows CRUD operations (Create, Read, Update, Delete)
  • Works well with languages like Java, Python, and PHP
  • Used in web apps, CMSs, and data-driven platforms

🚀 Why Use MySQL? 
  • Free, reliable, and fast
  • Easy to use for beginners and professionals
  • Widely used in small to large-scale applications
