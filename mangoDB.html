MangoDB

🍃 What is MongoDB?
    MongoDB is a popular NoSQL database that stores data in JSON-like documents instead of traditional tables, making it ideal for flexible and scalable applications.
        
🔧 Key Features: 
  • Document-oriented (uses BSON format)
  • Supports dynamic schemas (no need to predefine structure)
  • Great for big data, real-time analytics, and cloud apps
  • Works well with Node.js, Python, and other modern stacks

🚀 Why Use MongoDB?
  • Handles unstructured or changing data easily
  • Scales horizontally for large datasets
  • Used in modern web and mobile applications
